
# 🚀  Taller de Práctica con Git y GitHub

Repositorio Creado desde cero para empezar a dominar el flujo de trabajo con sistemas de control de versiones.

## 🛠️ Tecnologías utilizadas
* **Git** (Control de versiones local)
* **GitHub** (Plataforma de alojamiento en la nube)

## 📖 Lo que aprendimos en este ejercicio
1. A inicializar un repositorio local de forma manual (`git init`).
2. A conectar mi terminal con mi cuenta de GitHub remota.
3. El uso correcto de los comandos esenciales: `add`, `commit` y `push`.
4. La importancia de la rama principal `main` y la documentación del proyecto.
